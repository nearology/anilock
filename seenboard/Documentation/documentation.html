مقدمه

SeenBoard بسته ای کامل است که برای کمک به ایجاد یک رابط کاربری سفارشی و منحصر به فرد برای پروژه شما ساخته شده است.

با SeenBoard شما به ایجاد یک صفحه تنها برای پنل مدیریت خود محدود نمی شوید. شما بیش از بسیاری از قالب های پیش ساخته ساخته شده را انتخاب می کنید، بسیاری از سناریوهای محبوب و انواع محتوا را پوشش می دهید. مانند نسخه های اصلی مدیر، تمام قالب های صفحه قابل تنظیم هستند.

شما می توانید هر یک از عناصر UI از قبل ساخته شده را به طرح خود اضافه کنید. گزینه ها عظیم هستند، از جمله عناصر شامل دکمه ها، مدالها، ویجت ها، جدول زمانی، میله های پیشرفت و موارد دیگر. آنها همه می توانند سفارشی شوند تا اطمینان یابید که در صورت لزوم با سایر بخش های داشبورد شما سازگار هستند.

SeenBoard با صفحاتی همراه با کد مستندسازی شده به منظور ساده سازی روند سفارشی سازی ارائه شده است.

نویسنده: Mt.Rezaei در استودیو سپید هنر

SeenBoard همه مرورگرهای جدید را پیشتیبانی می کند (Chrome, Firefox, Safari) شامل Edge نیز هست .

نصب

SeenBoard یک قالب HTML استاتیک ساده است بنابراین به راحتی میتوانید آن را نصب کنید.

ساختار استفاده از قالب شامل مراحل ساده زیر است:

فایلهای HTML + assets

  |   |   └── all css files
  |   |
  |   └── fonts

  |   |
  |   └── icons
  |   |   └── all icons files
  |   |
  |   └── img
  |   |   └── all img files
  |   |
  |   └── js
  |   |   └── all javascript files
  |   |
  |   └── vendors
  |       └── all vendors css/javascript files
  |
  └── all html files

نسخه تیره

همه صفحات SeenBoard در نسخه تاریک در دسترس هستند.

فقط نام اسناد CSS را تغییر دهید: SeenBoard-1.0.css به SeenBoard-1.0-dark.css تغییر دهید.

همان چیزی که برای فایلهای جاوا اسکریپت وجود دارد: file.js به file-dark.js .

افزودن .min برای فایل های minified.
